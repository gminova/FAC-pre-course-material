COMMAND ACTION
ctrl + alt + t opens terminal
echo $0 confirms you are in Bash
pwd prints working directory
ls lists files
ls -l lists files with their permissions
ls -a lists all files, including hidden files
ls -al lists all files with their permissions, including hidden files
cd change directory to home directory
cd /subDirectoryName enters specific sub-directory
cd .. returns back to parent directory
pushd pushes directory
popd pops directory
mkdir directoryName creates new directory
mkdir -p folder1/folder2/folder3 creates new directory path
rmdir directoryName removes current empty directory
use -r for recursive removal
touch fileName.extension creates new file
cp fileName.extension copiedFileName.extension copies a file
cp fileName.extension directory_path copies a file in other directory
use -r for recursive copying
mv fileName.extension renamedFile.extension renames specific file
mv fileName.extension directory_path moves specific file to other directory
cat file1.txt file2.txt > file3.txt concatenates files
rm fileName.extension removes specific file
use -r for recursive removal
clear clears terminal window
locate fileName.extension finds files globally
find fileName.extension finds files within directory
grep find things inside files
hostname prints computer's network name
less pages through a file
xargs executes arguments
ping google.com checks internet speed
use (Ctrl + c) OR (Ctrl + Shift + c) to terminate
chmod change mode sets permissions of read/write/execute
for users, members and others
chown change owner
man commandName explains what a command does
apropos finds what manual page is appropriate
env looks at your environment
export exports/sets a new environment
exit exits terminal
